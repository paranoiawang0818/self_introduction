## 1. 网站定位

新版网站不建议做成传统简历，也不建议只做炫酷视觉主页。更适合定位为：

**一个面向同龄人、老师、学长学姐、合作伙伴和未来机会的成长型个人主页。**

它要回答四个问题：

1. 你是谁？
2. 你正在关注和投入什么？
3. 你有哪些可信证据？
4. 为什么别人值得和你连接？

## 2. 核心人物主线

推荐主线：

**王高是一个正在从高效学习者成长为 AI + 科研 + 自动化实践者的 XJTLU 本科生，同时具备公开表达、知识整理、项目落地和连接他人的能力。**

更短的对外表达：

**XJTLU 信息与计算科学本科生，关注 AI 工具、模型行为评测、自动化工作流与学习系统建设。**

更有个人感的表达：

**我正在用 AI、数学和自动化，把学习、研究和信息处理变成可复用的系统。**

## 5. 推荐页面叙事顺序

推荐顺序：

1. 首屏：我是谁，我关注什么
2. 核心快照：我有哪些可信证据
3. Research：我正在深入什么问题
4. Projects：我做出了什么东西
5. Public Communication：我如何表达和连接
6. What I Can Help With：我能给别人什么价值
7. Connect：我想遇见怎样的人

这个顺序比“个人信息 -> 服务 -> 生活照片 -> 想学习 -> GitHub”更适合新版网站，因为它先建立专业可信度，再释放人格和连接感。

## 6. 下一步进入 UI 前需要确定的问题

在开始写网站前，最好确认以下内容：

1. 新网站更偏“专业科研技术感”，还是“年轻个人 IP 感”？
2. 是否保留生活照片墙？如果保留，是放中段还是结尾？
3. 是否公开微信二维码？如果公开，是否只放在联系弹窗里？
4. 项目区是否需要每个项目配 GitHub 链接、截图或 Demo 链接？
5. 站点语言只做中文，还是中文为主、英文关键词辅助？
6. 是否希望网站未来支持持续更新，例如新增项目、文章、动态？

## 7. 当前建议

内容层已经接近可以进入 UI 设计阶段。

在进入开发前，下一步建议先确定 UI 方向，包括：

- 整体视觉气质
- 页面布局偏好
- 是否延续当前黑色代码宇宙风
- 是否需要照片、项目截图、动态背景
- 是否做单页滚动，还是多页面结构
- 是否需要移动端优先
